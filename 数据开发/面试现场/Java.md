47，编程：列出1-2000以内的，求53的余后为7的质数

```java
public static void main(String[]arg)
    {
        for(int i=2;i<=2000;i++)
            if (count(i))
                index(i);
    }
    public static boolean count(int n)
    {
        for(int j=2;j<n;j++)
        {
            if((n%j)==0)
                return false;
        }
        return true;
    }
    public static void index(int m)
    {
        if(53%m==7)
            System.out.print(m+" ");
    }

```
